# 学习规划

当用户提出"帮我规划复习/备考"类请求时，按以下步骤产出计划。

## 执行步骤

1. 先问清楚三件事：考试/截止日期、当前最薄弱的章节、每天可投入的时长。
2. 按"距离截止还有几周"做周级拆解，薄弱章节优先、靠前。
3. 把本周计划细化到每天，单日总时长不超过用户偏好上限（默认 3 小时）。
4. 用 todo 工具把当天任务建成清单，便于跟进。
5. 复盘时回顾错题与未完成项，滚动调整下一周计划。

## 工具建议

- todo：把每日任务落成清单
- calculator：按总时长/天数估算每日配额

## 检查清单

- [ ] 是否拿到截止日期与量化目标
- [ ] 薄弱章节是否被优先安排
- [ ] 单日时长是否在用户偏好范围内
- [ ] 是否建立了可跟进的 todo
